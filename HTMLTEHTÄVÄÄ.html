Tämä on kurssin ensimmäinenn htmlsivu

Kokeillaan html 5 koodin kirjoitusta < HTML> tägillä aloitetaan ja < /HTML> lopetetaan koko setti suurin osa tägeistä tarvitsee lopputägin.
< HEAD> < /HEAD> sisältää metadataa ja infoa, esim CSS tiedostonhakua etc
< BODY> < /BODY> on pääosuus sivusta
< P> < /P> on kappale
< IMG> tägillä laitetaan kuva (ei tarvitse lopputägiä)
< A> < /A>on hyperlinkki
< BR> < /BR> rivin vaihto
< H1-6> < /H1-6> tägit ovat erikokoisia otsikoita, 1 isoin 6 pienin
